# mweb-aiyouxi-guide

本仓库用于归档和发布多个独立 HTML 页面。页面内容均为静态资源，不针对任何特定域名或网站。

## 项目简介

mweb-aiyouxi-guide 是一个 HTML 页面归档仓库。仓库中的每个页面均为独立文件，不依赖外部服务器或数据库，可直接通过浏览器打开查看。

## 目录说明

```
.
├── README.md          # 本说明文件
├── index.html         # 索引页（如有）
└── pages/             # 归档页面存放目录
    ├── page1.html
    ├── page2.html
    └── ...
```

- `index.html`：可选的入口页面，用于展示或引导。
- `pages/`：存放所有归档的独立 HTML 页面。

## 页面归档说明

- 归档页面均为静态 HTML 文件，不包含后端逻辑。
- 每个页面独立完整，不依赖其他资源（CSS/JS 已内联或本地引用）。
- 页面内容可能包含文字、图片、链接等，均为归档时已固定的内容。
- 归档页面仅供个人参考或学习使用，不构成对任何外部服务的承诺或推荐。
